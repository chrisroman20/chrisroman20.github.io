¿Qué es?

El sensor de huellas dactilares AS608 es un sensor de huellas dactilares óptico de alta calidad y bajo costo. El sensor es pequeño y fácil de usar. La extracción de huellas dactilares, la imagen y la verificación se pueden realizar en un instante. El sensor puede almacenar hasta 200 huellas dactilares. El sensor se puede leer a través de UART con una velocidad de transmisión de 9600 bps, 19200 bps, 28800 bps, 38400 bps, 57600 bps, 115200 bps, 230400 bps, 460800 bps, 921600 bps, 1228800 bps. El sensor se puede usar con Arduino, Raspberry Pi, Beaglebone y otros microcontroladores.

Caracteristicas principales

  • Precisión

    Ofrece una alta precisión en la lectura de huellas, asegurando una autenticación confiable.

  • Velocidad

    Realiza verificaciones en segundos, ideal para aplicaciones en tiempo real.

  • Almacenamiento

    Puede guardar múltiples huellas, permitiendo la autenticación de varios usuarios.

  • Conectividad

    Se integra fácilmente en sistemas a través de interfaces comunes como UART, SPI o USB.

  • Tamaño Compacto

    Su diseño compacto es ideal para aplicaciones con espacio limitado.

  • Eficiencia Energética

    Es adecuado para dispositivos con batería debido a su bajo consumo de energía.

Aplicaciones Tipicas

  • Sistemas de seguridad

    Control de acceso a edificios, cajas fuertes, y sistemas de alarma.

  • Dispositivos Móviles

    Desbloqueo de teléfonos inteligentes y tabletas.

  • Aplicaciones Financieras

    Verificación de identidad para transacciones bancarias y financieras.
